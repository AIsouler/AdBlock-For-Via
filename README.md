## 说明

- 本仓库修改自上游仓库，移除混合规则全量版生成逻辑
- 规则每 **12 小时自动更新一次**
- 在上游 **混合规则精简版** 的基础上新增了几个规则源
- 仅包含 **Via浏览器** 支持的规则语法
- 若遇到误拦截或漏拦截问题，请向对应上游规则反馈
